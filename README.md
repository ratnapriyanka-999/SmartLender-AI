# 🚀 SmartLender AI - Loan Eligibility Prediction System

An AI-powered Loan Eligibility Prediction System that helps financial institutions assess loan applications using Machine Learning. The application predicts whether a loan is likely to be approved based on an applicant's financial and personal information through an intuitive web interface.

---

## 🌐 Live Demo

**Live Application:** https://smartlender-ai.onrender.com

---

## 📌 Project Overview

SmartLender AI automates the preliminary loan evaluation process by combining Machine Learning with a modern Flask web application.

Users can securely register, log in, submit loan application details, and receive an instant prediction generated by a trained Machine Learning model.

The project demonstrates the integration of Artificial Intelligence, Data Preprocessing, Web Development, and Database Management into a single application.

## 📂 Dataset

**Loan Prediction Dataset**

- Total Records: **614**
- Features: **17**
- Target Variable: **Loan Status**

Feature Engineering includes:

- Total Income
- Loan-Income Ratio
- Monthly Loan Burden

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/ratnapriyanka-999/SmartLender-AI.git
```

### Navigate to the Project Folder

```bash
cd SmartLender-AI
```

### Install Required Packages

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Open in Browser

```
http://127.0.0.1:5000
```
