# SmartLender AI

SmartLender AI is a Machine Learning-based Loan Eligibility Prediction System developed using Flask, Python, and XGBoost.

## Features

- User Registration and Login
- Loan Eligibility Prediction
- Machine Learning Model (XGBoost)
- Analytics Dashboard
- AI Model Information
- Responsive User Interface

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Bootstrap
- Scikit-learn
- XGBoost
- Pandas
- Joblib

## Dataset

Loan Prediction Dataset (614 records)

## Machine Learning

Algorithms Trained:

- Decision Tree
- Random Forest
- K-Nearest Neighbors
- XGBoost

Best Model:
**XGBoost**

Accuracy:
**82.93%**

## Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/SmartLender-AI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
